## Installation

1. Download the following files:
   - `themes/rust-rover.json`
   - `package.json`
2. Copy them to Zed's themes folder:
   ```bash
   mkdir -p ~/.zed/themes
   cp themes/rust-rover.json ~/.zed/themes/
   cp package.json ~/.zed/themes/
3. Restart Zed and select the theme.
